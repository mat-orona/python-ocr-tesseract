# OCR en Python con Tesseract

Herramienta básica de reconocimiento óptico de caracteres (OCR) desarrollada en Python utilizando Tesseract.

## Funcionalidades
- OCR en imágenes
- Soporte para idioma español
- Código modular
- Base para interfaz gráfica

## Requisitos
- Python 3
- Tesseract OCR
- pytesseract
- Pillow

## Uso
```bash
python main.py
